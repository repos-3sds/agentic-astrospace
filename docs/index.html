API Reference

Run locally with python main.py. Interactive docs at localhost:8000/docs.

MethodEndpointDescription
POST/api/v1/chartCalculate natal birth chart
POST/api/v1/readingFull AI natal chart reading
POST/api/v1/horoscopeDaily or weekly horoscope
POST/api/v1/compatibilitySynastry compatibility reading
POST/api/v1/transitsPersonal transit reading
GET/api/v1/transits/currentCurrent sky positions
Quick Start
# Install dependencies
pip install -r requirements.txt

# Configure API key
cp .env.example .env
echo "ANTHROPIC_API_KEY=your_key_here" >> .env

# Run the server
python main.py
# → API live at http://localhost:8000

# Example: Get a natal chart
curl -X POST http://localhost:8000/api/v1/chart \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Elon Musk",
    "year": 1971, "month": 6, "day": 28,
    "hour": 7, "minute": 30,
    "city": "Pretoria", "nation": "ZA"
  }'
